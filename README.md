# SigmaGPT 🤖
(https://sigma-gpt-amber.vercel.app/)

SigmaGPT is a modern AI chat application inspired by ChatGPT. It provides a clean interface for chatting with an AI assistant, managing conversation history, and securely handling user accounts.

##  Features

*  User Authentication (Login & Register)
*  AI Chat Interface
*  Chat History
*  Delete Chat Threads
*  Start New Chat
*  Fully Responsive Design
*  User Profile
*  Settings Page
*  Upgrade Plan Page
*  Protected Routes using JWT Authentication

##  Tech Stack

### Frontend

* React.js
* React Router DOM
* Context API
* CSS3
* React Spinners
* UUID

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

### AI

* Groq_AI API

##  Installation

Clone the repository

```bash
git clone https://github.com/Prince8382/SigmaGPT.git
```

Go to the project directory

```bash
cd SigmaGPT
```

Install frontend dependencies

```bash
npm install
```

Install backend dependencies

```bash
cd server
npm install
```
Start the frontend

```bash
npm run dev
```

Start the backend

```bash
npm start
```
